# 🪰 Mata-Mosca

Um jogo divertido de mata-mosca desenvolvido com HTML, CSS e JavaScript puro. Teste seus reflexos clicando nas moscas antes que o tempo acabe!

## 🕹️ Como jogar

1. Abra o jogo no navegador
2. Clique nas moscas que aparecem na tela para eliminá-las
3. Você tem vidas limitadas — não deixe nenhuma escapar!
4. Tente fazer a maior pontuação possível antes do tempo acabar

## 🚀 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/GabryLeo/Mata-mosca.git
```
2. Abra o arquivo `index.html` no seu navegador

Ou acesse pelo GitHub Pages: [Jogar agora 🎮](https://gabryleo.github.io/Mata-mosca/)

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 Estrutura

```
Mata-mosca/
├── index.html       # Tela inicial
├── app.html         # Tela do jogo
├── fim_de_jogo.html # Tela de game over
├── vitoria.html     # Tela de vitória
├── jogo.js          # Lógica do jogo
├── estilo.css       # Estilos
└── assets/          # Imagens (mosca, corações, etc.)
```

## 📚 Sobre o projeto

Projeto desenvolvido para praticar manipulação do DOM, eventos de clique, temporizadores com `setInterval` e lógica de jogo em JavaScript.
